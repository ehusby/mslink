This application allow you to create Windows Shortcut files (extension .LNK) without needing a Windows OS.

The work has been based on the documentation from:

Add new options:

-a, --argumentsSpecifies the command-line arguments
-i, --iconSpecifies the location of the icon to be used when displaying a shell link item in an icon view
